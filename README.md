# A web client for AV Zeus

## Development

```
$ yarn dev
```

A devvelopment server will start at [localhost:3000](http://localhost:3000).

## Deployment

```
$ amplify publish
```

A production server will start at [https://www.avzeus.net/](https://www.avzeus.net/).

## Libraries

- Next.js 12.0.2
- Redux 7.2.6
- antd 4.16.13
- amplify 4.3.4
